Browser Reporting

  • (Navigator) Version Info=
  • document.width= /* Mozilla says 'obsolete' */
  • document.body.clientWidth=
  • document.documentElement.clientWidth=
 
  • window.screen.width=
  • window.screen.height=
 
  • window.innerWidth=
  • window.innerHeight=
  • window.outerWidth=
  • window.outerHeight=

 

JQuery Reporting

  • JQuery version is:
 
  • $(window).innerWidth=
  • $(window).innerHeight=
  • $(window).outerWidth=
  • $(window).outerHeight=

 

Cordova Reporting

  • device.cordova=
  • device.model=
  • device.platform=
  • device.uuid=
  • device.version=
